powershell
.venv\Scripts\python.exe train.py   # auto-discovers new labels
Golden rule: Minimum 25 diverse phrasings per intent for good accuracy. Use generic words like "stock", "shares", "position" — not just specific ticker names.

Troubleshooting
Problem	Fix
Wrong intent	Add more paraphrase examples for that class, retrain
All ~14% confidence (random)	Batch size too large for data volume — use BATCH_SIZE=16
Misspelled / unknown stock name predicted wrong	Add generic-phrasing examples, not just ticker-specific ones
ModuleNotFoundError: torch	Use .venv\Scripts\python.exe, not system python
